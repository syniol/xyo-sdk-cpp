# C++ Mascot Generation Instructions

The `mascot.png` image in this directory was generated using an AI image generation tool to match the cohesive high-tech aesthetic of our other SDKs.

## Idea and Concept
The mascot is inspired by the internet legend of "Keith" (an obese, diseased rat missing a hind leg), which is famously referenced in developer humor surrounding C++. We reimagined Keith as a **sleek, cybernetic blue rat** featuring futuristic circuit board patterns. This design brings the legendary joke mascot into the unified, high-tech aesthetic established by the Rust, Node.js, PHP, and Golang SDK mascots, while projecting precision and stability. It also features the 'XYO Financial' plate.

## Recreation Details
* **Reference Images Used:** 
  * `../../rust/docs/rust_mascot_sleek_neon.jpg`
  * `../../php/docs/mascot.png`
* **Prompt Used:** The original image was generated with the prompt: "A futuristic neon illustration of an obese, cybernetic rat mascot missing a hind leg, representing C++. The rat has a sleek geometric design made of vibrant glowing neon purple circuit board patterns on a dark background." The hue of the neon glow was then programmatically shifted from purple to a sleek cybernetic blue to match the traditional C++ corporate aesthetic.
